## 🛠️ Capabilities (What We Do)

Don't just ask us to "write code". Ask us to **own outcomes**.

### 🏗️ Build
*   **"Scaffold a mechanic"** -> The Principal Architect will design the Database, Architecture, Events, and Scriptable Objects.
*   **"Create an Inventory Screen"** -> The UI Lead will build the intutive UI.

### 🔍 Review
*   **"Why is does this scene take so long to load?"** -> The Performance Engineer will analyze loading bottlenecks.
*   **"Check for regressions"** -> The QA Engineer will run the E2E suite.
*   **"Test authorization boundaries"** -> The Security Consultant will verify RLS and action-level controls.

### 🧠 Plan
*   **"Design a the architecture for Unique Skill Effects"** -> The System Architect will propose a strategy pattern to handle unique skill effects.

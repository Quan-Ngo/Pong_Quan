**Personal Twist**
During the course of the game, a powerup circle will appear in the middle of the field, moving up and down. The player who hits the circle with the ball will receive one of 5 powerups:
1. Paddle Lengthened
2. Faster Paddle Speed
3. Goal Guards appear at the 2 edges of of the goal
4. Slowing Field appears on their side
5. Power Strike increases ball velocity when their paddle hit the ball
I chose this personal twist for two reasons. The first is because I thought it will increase the excitement of the game by introducing a bit of randomn modifier to the game as well as give the players a secondary objective to aim for. The second is because I believe this mechanic demonstrats my knowledge of Unity and writing clean code with scalable and extensible architecture. I hope you enjoy my take on the classic game Pong.

I used an AI assisted workflow to help quickly build, iterate, and polish the project. I start by designing the game mechanics, then use my knowledge of Unity Engine to instruct the AI to write code that implements the mechanic, adhering to clean code architectures and principles. I then review the code that the AI generated and fix any issue I find.
